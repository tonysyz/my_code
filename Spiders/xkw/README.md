# xkwjzjxt_pojie 
***
手机端的学科网精准教学通会在点开试卷时请求一个包，特征是有"GetHomeWorkDetailed"的链接，后面的classid,homeworkid等等是其参数，只要按照这个请求就可以获取题目，答案，解析。用Fiddler抓获并把request文件保存到本地，和程序放在一个目录里面，进入程序并输入文件名即可获取答案。
